# CSS
**Navadni selector**
```css
body {
  background-color: lightblue;
}
```
**ID**
```css
#para1 {
  color: red;
}
```
**CLASS**
```css
.center {
  text-align: center;
}
```
**KOMBINACIJA**
```css
p.center { //<p> elements with class="center"
  text-align: center;
  color: red;
}
```
**ZA VSE**
```css
* {
  text-align: center;
  color: blue;
}
```
